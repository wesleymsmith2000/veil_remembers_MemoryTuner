## Suggested Placement

- Crystal ring radius: 26 studs
- Pylon ring radius: 42 studs
- Roof anchor radius: 13 studs
- Roof oculus height: about 52 studs

## Lane Layout

- Lane_01 = North / 12 o'clock
- Lane_02 = North-East / 1:30
- Lane_03 = East / 3 o'clock
- Lane_04 = South-East / 4:30
- Lane_05 = South / 6 o'clock
- Lane_06 = South-West / 7:30
- Lane_07 = West / 9 o'clock
- Lane_08 = North-West / 10:30

## Important

Roblox uses Y as the vertical/up axis. Later thread twisting should rotate mobile control nodes around the chamber's Y axis.

These meshes are not final art. They are scale/blockout assets for Codex to place, name, and connect to Beam/Attachment systems.
